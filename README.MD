# [AI FIRST DGS] Trilha de formação

Repositório para a trilha de formação do programa AI FIRST DGS, que tem como objetivo capacitar profissionais para atuarem com inteligência artificial generativa.

Cada branch do projeto corresponde a um cenário proposto da trilha de formação, contendo materiais de estudo, exercícios práticos e projetos para aplicação dos conhecimentos adquiridos.
